Research Paper Relation

By Kedar Kale

TF-IDF

  • Term Frequency - Inverse Document Frequency


    It is a numerical statistic that is intended to reflect how important a word is to a document in a collection.

    It is often used as a weighting factor in information retrieval, text mining, and user modeling.

Item based collaborative filtering

  • First, the system executes a model-building stage by finding the similarity between all pairs of items. This similarity function can take many forms, such as correlation between ratings


    Second, the system executes a recommendation stage. It uses the most similar items to a user's already-rated items to generate a list of recommendations. Usually this calculation is a weighted sum or linear regression.


    This form of recommendation is analogous to "people who rate item X highly, like you, also tend to rate item Y highly, and you haven't rated item Y yet, so you should try it".

KNN

  • K - Nearest Neighbours


    In pattern recognition, the k-nearest neighbors algorithm (k-NN) is a non-parametric method used for classification and regression.

    In both cases, the input consists of the k closest training examples in the feature space.

Input data collection and processing

By Pranav Jain

Our DataBase

We got our database from MovieLens. It contains more than 20 million ratings of 27,000+ movies. The data was created by 1,00,000+ users between January 09, 1995 and March 31, 2015.

Users were selected at random. All selected users had rated at least 20 movies.

We first plan to use a small-dataset with 1,00,000 ratings of several movies and then climb the ladder to the original dataset.

How we will process the data

We have already started working on our data. We have converted the csv files to sql format for easy handling and manipulation.

During each user's session, data will be retrieved from the database first and then using pandas, we'll format the data.
